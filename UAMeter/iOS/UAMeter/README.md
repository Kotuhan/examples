## Алгоритм визначення показників

1. Отримуємо картинку з камери
2. Використовуючи модель ScreenBoundaryML визначаємо чи є рамка лічильника у кадрі. Якщо є - повертаємо координати цієї рамки.
3. Виділяємо по координатам рамку екрану лічильника
4. За допомогою іншої моделі парсимо те, що є на екрані
   4.1) Визначаємо тип екрану
